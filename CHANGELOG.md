# سجل التغييرات (CHANGELOG)

جميع التغييرات الرئيسية على تطبيق **Walkie-Talkie LAN** موثقة هنا.

## [1.0.0] - 2026-01-09

### الابتكار والمميزات الجديدة
- **نظام PTT (اضغط للتحدث)**: تطبيق منطق الإرسال والاستقبال الصوتي الفوري.
- **بروتوكول UDP**: استخدام `RawDatagramSocket` لبث الصوت والتحكم عبر الشبكة المحلية لضمان أقل زمن تأخير.
- **اكتشاف الأجهزة تلقائياً**: إرسال نبضات (Presence broadcast) للتعرف على الأجهزة المتصلة دون الحاجة لـ IP يدوي.
- **إدارة الأدوار**: إضافة صلاحيات (Super Admin) للتحكم في المستخدمين الآخرين (كتم/حظر).
- **واجهة مستخدم عصرية**: تصميم مظلم (Dark Theme) مع تفاعل لوني (Amber/Red) يعبر عن حالات الاتصال.

### التحسينات البرمجية
- **دعم البناء السحابي**: إضافة `GitHub Actions` لبناء ملفات APK تلقائياً.
- **تحديث التوافق**: ترقية الحزم (`uuid`, `network_info_plus`, `wakelock_plus`) لأحدث الإصدارات المستقرة.
- **تحسين الأداء**: استخدام `const constructors` واستبدال الدوال المتهالكة مثل `withOpacity` بالمعيار الجديد `withValues`.
- **إدارة الطاقة**: دمج `WakelockPlus` لضمان عدم إطفاء الشاشة أو المعالج أثناء المحادثة.

### الإصلاحات (Fixes)
- إصلاح مشكلة فشل `flutter pub get` بسبب إصدارات الـ SDK المتعارضة.
- إصلاح تعليق التطبيق عند التشغيل على المتصفح (Web) بإضافة فحوصات المنصة.
- تحديث إصدارات GitHub Actions إلى `v4` لتجنب مشكلة الـ Deprecation.
- تصحيح أخطاء التهيئة في مكتبة `sound_stream`.
